# Gait Notes

## Early gait sequence

Start with a creep gait.

Only one leg lifts at a time.

Advantages:

- stable
- easy to debug
- lower current spikes
- easier to visualize

## Suggested order

1. Front Left
2. Rear Right
3. Front Right
4. Rear Left

## Motion phases

### 1. Lift
Raise knee slightly.

### 2. Swing
Move hip forward.

### 3. Plant
Lower knee.

### 4. Push
Move body forward.

## Debugging tips

- tune one leg first
- verify mirrored geometry
- slow all timing down initially
- test with chassis elevated
- watch servo current draw and heat
